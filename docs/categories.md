# Categories

Повний список категорій термінів у проєкті «Терміносфера».

---

### Git

Терміни систем контролю версій. Усе, що пов'язано з Git: команди, стратегії гілок, робочі
процеси, історія змін.

*Приклади:* repository, commit, branch, merge, rebase, clone, push, pull.

---

### DevOps

Автоматизація, CI/CD, інфраструктура, контейнеризація, оркестрація, моніторинг.

*Приклади:* deployment, continuous integration, Docker, Kubernetes, Ansible.

---

### Architecture

Архітектура програмного забезпечення: патерни проєктування, стилі архітектури, принципи
організації коду.

*Приклади:* cache, microservices, MVC, REST, event-driven architecture.

---

### Computer Science

Фундаментальні поняття інформатики: структури даних, алгоритми, теорія обчислень.

*Приклади:* algorithm, data structure, recursion, complexity, Big O notation.

---

### Frontend

Клієнтська розробка: браузерні технології, UI-фреймворки, веб-стандарти, доступність.

*Приклади:* DOM, CSS, HTML, React, accessibility, responsive design.

---

### Backend

Серверна розробка: API, бази даних, серверні фреймворки, бізнес-логіка.

*Приклади:* API, middleware, authentication, server, endpoint.

---

### Database

Системи керування базами даних, мови запитів, моделі даних, зберігання.

*Приклади:* SQL, NoSQL, index, migration, transaction, join.

---

### Security

Кібербезпека: захист даних, автентифікація, авторизація, шифрування, вразливості.

*Приклади:* encryption, authentication, CORS, XSS, SQL injection, OAuth.

---

### Network

Комп'ютерні мережі: протоколи, топології, маршрутизація, передача даних.

*Приклади:* TCP/IP, HTTP, DNS, latency, bandwidth, proxy.

---

### Testing

Тестування програмного забезпечення: методології, інструменти, підходи до перевірки якості.

*Приклади:* unit test, integration test, TDD, mocking, E2E, regression.

---

### Programming Languages

Мови програмування: синтаксис, парадигми, компілятори, інтерпретатори, типи.

*Приклади:* type system, compiler, interpreter, garbage collection, OOP, functional programming.

---

### Design

Дизайн інтерфейсів та взаємодії: UI/UX, типографіка, композиція, дизайн-системи.

*Приклади:* wireframe, prototype, typography, color theory, design system.

---

### Protocols

Мережеві та комунікаційні протоколи: формати, стандарти обміну даними.

*Приклади:* HTTP, HTTPS, WebSocket, GraphQL, gRPC, SMTP, FTP.

---

### Standards

Технічні стандарти та специфікації: міжнародні стандарти, RFC, специфікації W3C.

*Приклади:* ISO, RFC, ECMAScript, Unicode, WCAG, Semantic Versioning.

---

### AI/ML

Штучний інтелект та машинне навчання: нейронні мережі, алгоритми навчання, обробка даних.

*Приклади:* model, inference, prompt, embedding, token, neural network, LLM.

---

### Cloud

Хмарні обчислення: провайдери, сервіси, регіони, моделі розгортання.

*Приклади:* AWS, Azure, GCP, region, availability zone, serverless.

---

### Mobile

Мобільна розробка: платформи, фреймворки, формати пакунків, інструменти.

*Приклади:* APK, IPA, Flutter, React Native, SDK, Swift, Kotlin.

---

### Project Management

Управління проєктами: Agile, Scrum, планування, ролі, артефакти.

*Приклади:* backlog, sprint, epic, user story, stakeholder, retrospective.

---
