# Roadmap проєкту «Терміносфера»

Повний план розвитку проєкту. Актуальний статус завдань відстежується у
[ROADMAP.md](../ROADMAP.md) у корені репозиторію.

---

## v0.1 — MVP

**Мета:** створити працюючий застосунок із базовим пошуком, карткою терміна та перемиканням мов.

- SPA-роутинг (головна сторінка + сторінка терміна)
- Пошук за назвами та визначеннями трьома мовами
- Перемикання мов (EN / UK / ES)
- Світла та темна теми
- Адаптивний дизайн
- Білдер `@m00rl0ck/simple-builder`
- GitHub Pages

---

## v0.2 — Terminology Expansion

**Мета:** розширити термінологічну базу та покращити навігацію за категоріями.

- 100+ термінів із різних IT-галузей
- Фільтрація та навігація за категоріями
- Теги для термінів
- Покращений пошук (частковий збіг, транскрипція)
- Пов'язані терміни та кросс-посилання
- Автоматизована перевірка цілісності зв'язків

---

## v0.3 — Etymology

**Мета:** додати етимологічну інформацію для кожного терміна.

- Мова походження, корінь слова, значення кореня
- Історичні примітки для ключових термінів
- Фільтр «за мовою походження»
- Окремий розділ «Етимологія» на картці терміна

---

## v0.4 — Linguistic Features

**Мета:** розширити лінгвістичну інформацію про терміни.

- Частина мови
- Транскрипція / вимова (IPA)
- Словотвірні моделі (похідні слова)
- Транслітерація для українських та іспанських термінів
- Підсвічування спільних коренів між мовами
- Розділ «Мовні нотатки»

---

## v0.5 — Learning Mode

**Мета:** перетворити довідник на інструмент для вивчення IT-лексики.

### Personal Learning

- Позначати терміни як вивчені
- Прогрес користувача (localStorage)
- Рекомендовані терміни на основі вивченого
- Випадковий термін (кнопка "Мені пощастить")
- Термін дня (daily pick)
- Фільтр "показати невивчені"

### Flash Cards & Quizzes

- Флеш-картки (термін → переклад)
- Вікторини на вибір правильної відповіді
- Тематичні добірки термінів
- Режими «Вивчення IT-англійської» та «Вивчення IT-іспанської»
- Прогрес вивчення (localStorage)
- Щоденна порція термінів (daily dose)

---

## v0.6 — Community Features

**Мета:** залучити спільноту до наповнення та покращення бази.

- Форма пропозиції нового терміна
- Голосування за переклади
- Обговорення термінів (коментарі)
- GitHub Issues як бекенд для пропозицій
- Рейтинг термінів
- Сторінка «Останні додані терміни»

---

## v0.7 — AI Features

**Мета:** використати AI для покращення контенту та користувацького досвіду.

- Пояснення термінів для новачків (AI)
- Автоматичні приклади використання
- Пошук семантично схожих понять
- AI-асистент із термінології (чат-інтерфейс)
- Автоматичне зіставлення перекладів
- Валідація етимології через AI

---

## v1.0 — Stable Release

**Мета:** стабільна, повнофункціональна версія з відкритим API.

- 500+ верифікованих термінів
- Повна підтримка EN / UK / ES
- Етимологія для всіх термінів
- Навчальний режим (флеш-картки, вікторини)
- REST API для зовнішніх застосунків
- Документація API
- E2E-тестування
- CI/CD для GitHub Pages
- Lighthouse 90+
- a11y (WCAG AA)
- Стабільна документація
